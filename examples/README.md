# 示例代码使用说明

这些示例不是“完整项目”，而是配合学习计划使用的最小样例。

建议顺序：

1. `python/01_text_tool.py`
2. `python/02_http_json.py`
3. `python/03_fastapi_app/main.py`
4. `go/01_struct_and_slice.go`
5. `go/02_http_server.go`
6. `ai/01_chat_api/main.py`

学习方法：
- 先运行
- 再逐行读
- 再自己改
- 最后自己从空白文件重写一次

建议你每个示例至少做两次改动，例如：
- 新增一个参数
- 修改输出结构
- 新增一个函数
- 增加错误处理
- 新增一个接口
